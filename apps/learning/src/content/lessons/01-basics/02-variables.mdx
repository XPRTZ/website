## Exercise: Personal Information

Create a program that stores and displays personal information using different data types:
1. Name (string)
2. Age (integer)
3. Height in meters (floating-point)
4. Is a student (boolean)

Print all the information in a formatted way.

<Collapsible title="C# solution">
```csharp
using System;

class Program
{
    static void Main()
    {
        string name = "Alice";
        int age = 20;
        double height = 1.75;
        bool isStudent = true;

        Console.WriteLine($"Name: {name}");
        Console.WriteLine($"Age: {age}");
        Console.WriteLine($"Height: {height}m");
        Console.WriteLine($"Student: {isStudent}");
    }
}
```
</Collapsible>
<Collapsible title="Python solution">

```python
name = "Alice"
age = 20
height = 1.75
is_student = True

print(f"Name: {name}")
print(f"Age: {age}")
print(f"Height: {height}m")
print(f"Student: {is_student}")
```

</Collapsible>
<Collapsible title="Go solution">

```go
package main

import "fmt"

func main() {
    name := "Alice"
    age := 20
    height := 1.75
    isStudent := true

    fmt.Printf("Name: %s\n", name)
    fmt.Printf("Age: %d\n", age)
    fmt.Printf("Height: %.2fm\n", height)
    fmt.Printf("Student: %v\n", isStudent)
}
```
