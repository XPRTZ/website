## Getting Started with C#, Python, and Go

This guide will help you set up your development environment for all three languages.

### Exercise: Hello, World!

Write a program in each language that prints "Hello, World!" to the console. Try to do this without looking at the solution first!

<Collapsible title="C# example" defaultOpen={true}>
```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, World!");
    }
}
```
</Collapsible>

<Collapsible title="Python example" defaultOpen={true}>
```python
print("Hello, World!")
```
</Collapsible>
<Collapsible title="Go example" defaultOpen={true}>
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
</Collapsible>

### Exercise: Command Line Arguments

Write a program that accepts a name as a command line argument and prints "Hello, [name]!". For example, if the user runs the program with "Alice" as an argument, it should print "Hello, Alice!".

<Collapsible title="C# solution">
```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        string name = args.Length > 0 ? args[0] : "World";
        Console.WriteLine($"Hello, {name}!");
    }
}
```
</Collapsible>
<Collapsible title="Python solution">
```python
import sys

name = sys.argv[1] if len(sys.argv) > 1 else "World"
print(f"Hello, {name}!")
```
</Collapsible>
<Collapsible title="Go solution">
```go
package main

import (
    "fmt"
    "os"
)

func main() {
    name := "World"
    if len(os.Args) > 1 {
        name = os.Args[1]
    }
    fmt.Printf("Hello, %s!\n", name)
}
```
</Collapsible>
