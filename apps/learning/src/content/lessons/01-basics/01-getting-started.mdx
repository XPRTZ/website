## Getting Started with C#, Python, and Go

This guide will help you set up your development environment for all three languages.

### Setup development environment

This guide will be using VS Code and Windows Subsystem for Linux version 2.

#### C#

Install the .NET SDK in WSL: add the Microsoft package signing key and repository.
For Ubuntu (adjust based on your Ubuntu release; here we assume Ubuntu 24.04):
```bash
sudo apt install -y apt-transport-https ca-certificates wget
wget https://packages.microsoft.com/config/ubuntu/24.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
```

Update package lists and install the .NET SDK (use the version you need; here we install the latest version):
```bash
sudo apt update
sudo apt install -y dotnet-sdk-9.0
```

You can verify the installation with:
```bash
dotnet --version
```

Visual Studio Code extensions:
- [C# Dev Kit](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csdevkit)
- [IntelliCode for C# Dev Kit](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.vscodeintellicode-csharp)

#### Python

In the WSL terminal, run:
```bash
sudo apt update
sudo apt install -y python3 python3-pip
```

Verify the installation:

```bash
python3 --version
pip3 --version
```

Set up a virtual environment:
Install the venv module if required (most distributions include it):
```bash
python3 -m venv myenv
```

Activate the environment:
```bash
source myenv/bin/activate
```

Visual Studio Code extensions:
- [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
- [Python Debugger](https://marketplace.visualstudio.com/items?itemName=ms-python.debugpy)

#### Go

In the WSL terminal, run:
```bash
sudo apt update
sudo apt install -y golang-go
```

Verify the installation:

```bash
go version
```

Visual Studio Code extensions:
- [Go](https://marketplace.visualstudio.com/items?itemName=golang.Go)

### Exercise: Hello, World!

Write a program in each language that prints "Hello, World!" to the console. Try to do this without looking at the solution first!

### Exercise: Command Line Arguments

Write a program that accepts a name as a command line argument and prints "Hello, [name]!". For example, if the user runs the program with "Alice" as an argument, it should print "Hello, Alice!".

<Collapsible title="C# solution">
```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        string name = args.Length > 0 ? args[0] : "World";
        Console.WriteLine($"Hello, {name}!");
    }
}
```
</Collapsible>
<Collapsible title="Python solution">
```python
import sys

name = sys.argv[1] if len(sys.argv) > 1 else "World"
print(f"Hello, {name}!")
```
</Collapsible>
<Collapsible title="Go solution">
```go
package main

import (
    "fmt"
    "os"
)

func main() {
    name := "World"
    if len(os.Args) > 1 {
        name = os.Args[1]
    }
    fmt.Printf("Hello, %s!\n", name)
}
```
</Collapsible>
