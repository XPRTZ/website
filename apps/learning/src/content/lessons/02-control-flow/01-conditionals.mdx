# Conditional Statements

Learn how to control program flow using conditional statements.

<Collapsible title="C# example" defaultOpen={true}>

```csharp
int age = 18;

if (age >= 18)
{
    Console.WriteLine("You are an adult");
}
else if (age >= 13)
{
    Console.WriteLine("You are a teenager");
}
else
{
    Console.WriteLine("You are a child");
}
```

</Collapsible>
<Collapsible title="Python example" defaultOpen={true}>

```python
age = 18

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")
```
</Collapsible>
<Collapsible title="Go example solution" defaultOpen={true}>

```go
age := 18

if age >= 18 {
    fmt.Println("You are an adult")
} else if age >= 13 {
    fmt.Println("You are a teenager")
} else {
    fmt.Println("You are a child")
}
```
</Collapsible>

Each language has its own syntax for conditional statements while maintaining similar logical structures.
